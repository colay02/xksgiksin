适配15.10

## 使用方式

1. 使用**管理员模式**运行。
2. 锁定英雄时自动/手动在软件内选择英雄。
3. 选择皮肤。
4. 启动（进游戏之后可以关掉）。

## 打包

```shell
pyinstaller main.spec
pyinstaller main_with_debug.spec
```
